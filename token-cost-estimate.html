Bottom line

Most likely range

Building this project with modern AI coding assistance most likely consumed ~8–12 million tokens and cost roughly $54.00 (≈ 1,393,200 ₫) on pure API pricing, or about $26.00 (≈ 670,800 ₫) if absorbed entirely inside a Cursor Pro subscription. Realistic full range is $21.60 – $155.76 (≈ 557k – 4.0M ₫) depending on how often Opus-class models and heavy iteration were used.

Assumptions & caveats
  • Token count is inferred from final repo size + git churn — the real AI session traffic is not logged in this workspace.
  • Pricing uses Claude Sonnet 4-class rates ($3/M in, $15/M out). GPT-5 / Gemini would be 30–60% cheaper; pure Opus 4 would be ~5× more expensive.
  • Exchange rate assumed: 1 USD = 25,800 VND (mid-2026 estimate). Use your bank's spot rate for exact VND.
  • Does not include human review time, infrastructure, or any non-AI tooling cost.
